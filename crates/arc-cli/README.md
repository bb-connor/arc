# arc-cli

`arc-cli` is the operator-facing ARC binary crate. It drives the supported
entry paths in this repo: `arc check`, `arc mcp serve`, `arc trust serve`,
receipt inspection, and the local qualification loops used by the quickstarts.

Use `arc-cli` when you need the `arc` binary or the repo's supported runtime
shell. Start with:

- `../../docs/install/BINARY_DISTRIBUTION.md`
- `../../docs/guides/MIGRATING-FROM-MCP.md`
- `../../docs/guides/WEB_BACKEND_QUICKSTART.md`
