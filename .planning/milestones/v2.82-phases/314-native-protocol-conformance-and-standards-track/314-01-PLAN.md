# Plan 314-01

1. Add a dedicated native conformance suite with JSON scenario files covering
   the required categories from the roadmap.
2. Add a runner inside `arc-conformance` that can execute those scenarios via
   language-neutral `artifact`, `stdio`, and `http` driver contracts.
3. Add a deterministic fixture target and integration tests so the native suite
   is executable in-repo rather than existing only as paper design.
