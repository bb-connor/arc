# Plan 311-03

1. Check in versioned JSON Schema files under `spec/schemas/arc-wire/v1` for

   against the checked-in schemas.
3. Keep the schemas intentionally aligned to the shipped Rust surface rather
   than inventing future protocol variants or speculative hosted JSON-RPC
   message types.
