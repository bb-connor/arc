# Milestone Audit: v2.52 MERCURY Extensions ARC Purity Restoration, Boundary Cleanup, and Qualification

## Status

Milestone `v2.52` meets its implementation definition of done. Phases `217`
through `220` are implemented, verified, and restore the repo's intended
product boundary: ARC generic crates no longer own Mercury- or ARC-Wall-
specific packaging or receipt-query logic, and future Mercury work can proceed
only on Mercury's own app surface.

The resulting posture is specific:

  or generic CLI crates

  only filters

  Mercury-only receipt index
- the repo-wide non-product-crate audit returns no Mercury or ARC-Wall
  references outside dedicated app crates

  generic receipt-query tests
