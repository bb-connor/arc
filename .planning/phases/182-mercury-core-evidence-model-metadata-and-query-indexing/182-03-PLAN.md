# Plan 182-03

Extend ARC receipt-query surfaces so MERCURY investigation workflows can query
business identifiers and approval state through local CLI and trust-control
entrypoints.

## Scope

- add MERCURY filter fields to `ReceiptQuery`
- wire the new filters through SQLite query execution
- expose the same filters through `arc receipt list` and trust-control HTTP

## Success Criteria

- local receipt queries can filter by MERCURY business identifiers and
  approval state
- trust-control uses the same query contract as the local store
- regression tests cover the primary investigation filters for MERCURY
