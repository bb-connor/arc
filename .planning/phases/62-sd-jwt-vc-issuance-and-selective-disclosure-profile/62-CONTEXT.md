# Phase 62: SD-JWT VC Issuance and Selective Disclosure Profile - Context

## Goal

Add one standards-native SD-JWT VC issuance profile with bounded selective
disclosure over the external ARC passport projection.

## Why This Phase Exists

Portable credential issuance is not complete until ARC can mint one
standards-native credential that external wallets and verifiers can actually
validate.

## Scope

- SD-JWT VC issuance profile over OID4VCI
- holder binding and disclosure policy
- claim catalog and disclosability rules

## Out of Scope

- OID4VP verifier-side request transport
- non-SD-JWT proof families
- public wallet discovery
