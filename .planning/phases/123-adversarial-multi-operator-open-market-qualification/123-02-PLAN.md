# Plan 123-02

Prove visibility-versus-trust separation under adversarial conditions.

## Scope

- tests where malicious or stale public data remains visible but untrusted
- rejection cases for incompatible reputation, governance, or economic artifacts
- verification that local trust activation remains authoritative

## Success Criteria

- visibility never becomes trust under adversarial pressure
- malicious or stale external data fails closed
- local activation rules remain intact across the matrix
