# Phase 86: Delegation Bonds and Autonomy Tier Gates - Context

## Goal

Bind reserve-backed bond state into delegation and autonomy-tier gates so
economically sensitive execution fails closed without prerequisites.

## Why This Phase Exists

Bond state only matters if ARC can actually use it to permit or deny more
autonomous execution. The research's bonded-agent thesis requires explicit,
reserve-backed gates on higher-risk behavior.

## Scope

- delegation-bond attachment semantics
- autonomy-tier prerequisites
- reserve plus assurance gating behavior
- fail-closed execution denial reasons

## Out of Scope

- full loss and delinquency lifecycle
- provider claim workflows
- public marketplace governance
