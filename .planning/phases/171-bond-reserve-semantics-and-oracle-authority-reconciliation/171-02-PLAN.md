# Plan 171-02

Choose and implement one canonical oracle authority path across `arc-link`,
kernel, and contracts.

## Scope

- make the supported FX authority explicit in the typed oracle-evidence
  envelope
- keep `arc-link` as the only runtime authority for conversion evidence
- expose settlement-side oracle authority and optional price-resolver
  reference wiring through config and public docs

## Success Criteria

- receipt-side FX evidence always names one supported authority model
- settlement config records the authority choice instead of leaving it ambient
- public docs describe `ArcPriceResolver` as auxiliary and `arc-link` as the
  runtime authority
