# Phase 117: Generic Listing Artifact and Namespace Model - Context

## Goal

Generalize ARC's curated public discovery surfaces into one generic listing and
namespace model for tools, issuers, verifiers, providers, and future market
actors.

## Why This Phase Exists

The full endgame needs an open registry substrate, but it must start with a
portable listing model rather than ad hoc type-specific publication flows.

## Scope

- generic listing artifact shape
- namespace ownership and publication rules
- separation between visibility, identity, and trust activation
- fail-closed handling for conflicting or unverifiable listings

## Out of Scope

- origin or mirror ranking behavior
- open admission classes and trust activation artifacts
- portable reputation or marketplace-fee economics
