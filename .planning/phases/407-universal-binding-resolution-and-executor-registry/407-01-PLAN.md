# Phase 407 Plan

## Goal

Remove edge-local authoritative target branching and move claim-eligible target
resolution onto one shared executor-registry contract.

## Work Plan

1. Extend `arc-cross-protocol` with a reusable `TargetProtocolRegistry` that
   can resolve authoritative bindings without per-edge branch tables.
2. Generalize `CrossProtocolOrchestrator` to use the shared registry for
   non-native target execution instead of edge-local `match` statements.
3. Add at least one additional target family beyond `Native` and `Mcp`.
4. Migrate A2A and ACP publication/runtime resolution onto the shared registry.
5. Add regression tests proving protocol-aware target binding works through the
   edge default authoritative path.

## Verification

- `cargo test -p arc-cross-protocol -p arc-a2a-edge -p arc-acp-edge`
