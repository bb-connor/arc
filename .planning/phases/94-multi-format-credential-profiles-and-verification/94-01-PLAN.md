# Plan 94-01

Add a shared projection engine for multiple portable credential profiles.

## Scope

- support the current ARC SD-JWT VC projection plus one additional standards-legible profile
- keep all projections rooted in one passport truth and one signing policy surface
- make profile support explicit in issuer metadata and operator configuration

## Success Criteria

- ARC projects more than one portable credential profile from one truth source
- profile support is explicit rather than inferred
- projection behavior is deterministic and replayable
