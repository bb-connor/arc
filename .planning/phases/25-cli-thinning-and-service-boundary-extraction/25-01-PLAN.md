# Phase 25 Plan 01

## Objective

Define the public APIs and compatibility facades for the new service crates,
and isolate the shared support boundary that currently makes a direct move out
of `pact-cli` unsafe.

## Must Haves

- a concrete dependency map for `trust_control.rs`, `remote_mcp.rs`, and the
  helper functions still living in `main.rs`
- a chosen shared-support boundary that prevents crate cycles and avoids large
  code duplication
- updated phase context and research artifacts that justify the extraction
  order
