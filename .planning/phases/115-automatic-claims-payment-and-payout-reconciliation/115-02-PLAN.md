# Plan 115-02

Define reconciliation and fail-closed behavior for automatic payouts.

## Scope

- intended versus externally reconciled payout state
- stale authority, duplicate payout, and amount mismatch handling
- sidecar reconciliation truth for payout flows

## Success Criteria

- automatic payouts reconcile without mutating canonical claim truth
- stale or mismatched payout state fails closed
- reconciliation behavior is explicit for operators and auditors
