# Plan 157-03

Add fail-closed precondition checks for unsafe or unsupported dispatch.

## Scope

- validate binding purpose, chain scope, operator address, and instruction
  rail metadata before submission
- reject unsafe amount-scaling and malformed address inputs
- avoid RPC-default send behavior by estimating gas inside the runtime

## Success Criteria

- unsupported dispatch attempts are denied before submission
- decimal or identity mismatches fail closed
- bounded send semantics are deterministic across the runtime lane
