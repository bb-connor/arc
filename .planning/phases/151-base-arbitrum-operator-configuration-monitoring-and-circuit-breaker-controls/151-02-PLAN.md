# Plan 151-02

Build monitoring, alerting, and degraded-mode behavior for the oracle runtime.

## Scope

- add structured chain and pair health reports
- surface fallback, stale, degraded, paused, and unavailable states explicitly
- implement bounded stale-cache degraded mode with conservative margin

## Success Criteria

- drift and outage behavior is visible through one runtime report shape
- degraded-mode behavior is explicit, bounded, and disabled by default
- sequencer or oracle outages do not look like normal healthy reads
