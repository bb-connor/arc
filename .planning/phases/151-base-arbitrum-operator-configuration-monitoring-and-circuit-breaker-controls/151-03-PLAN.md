# Plan 151-03

Implement circuit-breaker controls and operator override semantics.

## Scope

- add global pause and pair-level disable controls
- support deterministic backend-selection overrides and fallback suppression
- record override state in runtime health output and fail closed in oracle reads

## Success Criteria

- operators can stop unsafe cross-currency enforcement deterministically
- override state is visible in monitoring output
- the kernel sees explicit oracle errors instead of silently widening trust
