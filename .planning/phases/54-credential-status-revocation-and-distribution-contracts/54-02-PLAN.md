# Plan 54-02

Implement lifecycle publication and query wiring across the existing passport
and trust-control surfaces.

## Scope

- extend the lifecycle registry and/or publication helpers to emit the new
  portable status projection
- expose local CLI and trust-control routes for publishing and resolving the
  portable lifecycle view
- thread distribution hints into the issued-passport flow where needed so
  holders and verifiers can discover lifecycle state

## Success Criteria

- operators can publish and query portable lifecycle state locally and remotely
- remote trust-control exposes holder/verifier-safe read surfaces without
  widening operator authority
- stale, missing, or contradictory lifecycle state fails closed
