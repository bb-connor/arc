# Plan 184-01

Build the first executable replay/shadow pilot harness on top of the frozen
MERCURY package contracts.

## Scope

- define the gold propose -> approve -> release -> inquiry scenario plus a
  rollback variant in `arc-mercury-core`
- add `arc mercury pilot export` to generate the pilot corpus end to end
- emit ARC evidence packages, proof packages, inquiry packages, and verification
  reports as deterministic pilot artifacts

## Success Criteria

- the primary workflow exports signed receipts, checkpoints, a proof package,
  and an inquiry package
- the rollback variant exports a proof package from the same workflow contract
- one command produces the full gold corpus and machine-readable summary
