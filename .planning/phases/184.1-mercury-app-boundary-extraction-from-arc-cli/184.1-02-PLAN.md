# Plan 184.1-02

Remove MERCURY-specific command wiring and MERCURY-only dependency edges from
generic ARC crates.

## Scope

- delete the `mercury` command family from `arc-cli`
- remove MERCURY-only crate dependencies that exist only because of the old
  CLI coupling
- move the Mercury integration coverage out of `arc-cli` and into the Mercury
  app package

## Success Criteria

- `arc-cli` no longer exposes a MERCURY command surface
- `arc-cli` and `arc-control-plane` do not carry MERCURY-only dependencies
- the Mercury CLI integration tests live with the Mercury app crate
