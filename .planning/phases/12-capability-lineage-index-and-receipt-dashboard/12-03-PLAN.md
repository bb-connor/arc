<!-- API contracts the dashboard calls (from Phase 10 + Plan 12-02) -->

GET /v1/receipts/query?capabilityId=&toolServer=&toolName=&outcome=&since=&until=&minCost=&maxCost=&agentSubject=&cursor=&limit=
Response: { totalCount: number, nextCursor: number | null, receipts: Receipt[] }

GET /v1/lineage/:capability_id
Response: { capability_id, subject_key, issuer_key, issued_at, expires_at, grants_json, delegation_depth, parent_capability_id }

GET /v1/lineage/:capability_id/chain
Response: CapabilitySnapshot[]  (root-first order)

GET /v1/agents/:subject_key/receipts?cursor=&limit=&outcome=&...
Response: { totalCount, nextCursor, receipts }

All endpoints require: Authorization: Bearer <token>

Receipt shape (from PactReceipt JSON):
{
  id: string,
  timestamp: number,
  capability_id: string,
  tool_server: string,
  tool_name: string,
  action: { parameters: object, parameter_hash: string },
  decision: "allow" | { deny: { reason, guard } } | { cancelled: {...} } | { incomplete: {...} },
  metadata?: { financial?: { grant_index, cost_charged, currency, budget_remaining, budget_total, delegation_depth, root_budget_holder, settlement_status } }
}

- Props: `filters: Filters`
- Uses `useReactTable` from @tanstack/react-table with `manualPagination: true`
- State: `receipts`, `totalCount`, `cursor` (number | null), `cursorStack` (for back navigation), `loading`, `selectedReceipt` (for detail panel)
- Fetches receipts via `fetchReceipts(filters, cursor, 50)` on filter or cursor change (useEffect)
- Column definitions via `createColumnHelper<Receipt>()`:
  - Time: formatted via date-fns `format(new Date(receipt.timestamp * 1000), 'yyyy-MM-dd HH:mm:ss')`
  - Tool: `${tool_server}/${tool_name}`
  - Decision: badge-styled "Allow" / "Deny" / etc.
  - Capability ID: truncated to 8 chars with full ID on hover
  - Cost: monetary amount from metadata.financial.cost_charged formatted as `(cost / 100).toFixed(2)` with currency prefix, or "--" if no financial metadata
- Pagination controls: "Previous" / "Next" buttons. "Next" pushes current cursor to stack and sets cursor = nextCursor. "Previous" pops cursorStack. Disable Previous when stack is empty, disable Next when nextCursor is null.
- Row click opens detail panel (sets selectedReceipt)
- Detail panel shows full receipt JSON, delegation chain expandable (uses DelegationChain component), and budget info if financial metadata present

- Use lucide-react ChevronDown/ChevronRight icons for expand/collapse
- Loading state while fetching
