# Plan 172-02

Move the official contract and bindings surface toward generated or
artifact-derived interfaces.

## Scope

- repair interface drift in the Solidity interface layer
- derive `arc-web3-bindings` from compiled interface artifacts
- provide shared proof-type adapters for detailed Merkle-proof calls

## Success Criteria

- Rust bindings compile from the shipped interface artifacts instead of a
  manually duplicated ABI surface
- Solidity interface artifacts and implementation artifacts expose the same
  intended callable surface
- detailed Merkle-proof calls remain ergonomic at runtime without manual ABI
  redefinition
