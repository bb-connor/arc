# Plan 142-02

Define discovery semantics and lookup behavior for public wallets.

## Scope

- define lookup, refresh, and freshness behavior
- document how discovery remains informational rather than trust-granting
- record failure modes for stale, conflicting, or unsupported routing data

## Success Criteria

- wallet discovery semantics are explicit
- discovery does not silently widen trust
- stale or conflicting routing data fails closed
