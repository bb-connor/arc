# Phase 142: Public Wallet Directory, Routing, and Discovery Semantics - Context

## Goal

Define wallet directory, routing, and discovery semantics for the broadened public identity network.

## Why This Phase Exists

Once methods are broader, ARC needs one explicit public wallet routing layer rather than ad hoc discovery behavior.

## Scope

- wallet directory artifacts
- routing semantics
- discovery metadata
- trust-preserving lookup behavior

## Out of Scope

- full interop qualification
- final public-boundary closure
- new trust-federation mechanics
