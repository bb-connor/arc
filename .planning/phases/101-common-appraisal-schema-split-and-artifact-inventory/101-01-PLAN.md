# Plan 101-01

Define the common appraisal artifact model and schema split.

## Scope

- typed portable appraisal artifact over existing ARC appraisal truth
- explicit split between raw evidence, verifier metadata, normalized assertions, and policy-facing conclusions
- schema versioning and compatibility boundaries

## Success Criteria

- ARC has one explicit outward-facing appraisal artifact contract
- artifact layers do not collapse raw evidence and normalized claims
- versioning rules are explicit and auditable
