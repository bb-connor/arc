# Phase 130: Unified Contract Interfaces, Bindings, and Chain Configuration - Context

## Goal

Define the unified on-chain contract set, ARC bindings, and official chain configuration for the first web3 execution stack.

## Why This Phase Exists

After the boundary is frozen, ARC needs one canonical contract surface and deployment model before anchoring or settlement logic can land.

## Scope

- unified contract interfaces
- generated bindings
- official chain configuration
- deployment and gas assumptions

## Out of Scope

- root anchoring lifecycle
- oracle evidence ingestion
- settlement execution flows
