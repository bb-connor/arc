# Plan 46-01

## Objective

Add the adapter-facing contract and durable persistence for post-execution
metered-cost evidence without mutating signed receipts.

## Work

- Define stable metered-evidence and reconciliation types for operator/report
  surfaces.
- Add SQLite sidecar storage keyed by `receipt_id` with replay-safe uniqueness
  over external evidence identifiers.
- Add validation rules for empty identifiers, zero units, zero costs, and
  evidence records attached to receipts that do not carry metered-billing
  governed context.

## Verification

- targeted unit coverage for normalization, limit clamping, and action-needed
  logic
- store-level regression coverage for replay rejection and invalid attachment
  paths
