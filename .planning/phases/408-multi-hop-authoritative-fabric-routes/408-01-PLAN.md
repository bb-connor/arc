# Phase 408 Plan

## Goal

Prove real shared-fabric multi-hop authoritative execution with continuous
trace/route evidence across at least two qualified flows.

## Work Plan

1. Extend target executors to return explicit route hops instead of only a
   terminal projection.
2. Teach `CrossProtocolOrchestrator` to assemble route evidence and trace hops
   from executor-provided hop data.
3. Model MCP-mediated routes as explicit multi-hop `source -> mcp -> native`
   authoritative paths in the shared fabric.
4. Add edge assertions proving A2A and ACP authoritative routes carry
   multi-hop route evidence.
