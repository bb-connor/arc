# Plan 305-02

## Objective

Flip the raw kernel tool-call API to async `&self` and preserve compatibility
for the still-synchronous session-backed and CLI/runtime callers.

## Tasks

1. Make `ArcKernel::evaluate_tool_call` an `async fn` on `&self`.
2. Add a narrow synchronous compatibility path for legacy sync callers.
3. Move session-backed operational methods to `&self` so `arc-mcp-edge` and
   the CLI no longer depend on exclusive kernel ownership.

## Acceptance

- The public raw tool-call API is async and shared-access.
- `arc-mcp-edge`, CLI session flows, and control-plane wiring still compile.
- The migration does not introduce a coarse kernel-wide serialization lock.
