<objective>
Add optional guest export detection for arc_alloc (WGRT-06) and
arc_deny_reason (WGRT-07) to the WasmtimeBackend evaluate() flow, with
safe fallbacks when the exports are absent or return invalid values.

Purpose: Enables guests with allocators to control where request data is
written in linear memory, and enables guests to return structured deny
reasons. Both features gracefully degrade for simpler guests.

Output: Updated evaluate() in runtime.rs with arc_alloc probing before
memory write and arc_deny_reason probing after VERDICT_DENY; comprehensive
WAT-based unit tests for all code paths.

    1. **Define the arc_deny_reason protocol.** The guest exports
       `arc_deny_reason(buf_ptr: i32, buf_len: i32) -> i32` which the host
       calls with a pointer to a host-provided buffer region in guest memory.
       The guest writes a JSON-encoded `GuestDenyResponse` into that buffer and
       returns the number of bytes written (or -1 on error).

       Implementation approach: use a fixed buffer region at offset 64K (same
       as the legacy region) with a max length of 4096 bytes. The host calls
       `arc_deny_reason(65536, 4096)` and reads back the response.

    2. **After `VERDICT_DENY` in evaluate()**, replace the current
       `let reason = read_deny_reason(&memory, &store);` with:

       ```rust
       crate::abi::VERDICT_DENY => {
           // Probe for structured arc_deny_reason export
           let deny_reason_fn = instance
               .get_typed_func::<(i32, i32), i32>(&mut store, "arc_deny_reason")
               .ok();

           let reason = if let Some(ref reason_fn) = deny_reason_fn {
               read_structured_deny_reason(reason_fn, &memory, &mut store)
           } else {
               // Fallback to legacy offset-64K NUL-terminated string
               read_deny_reason(&memory, &store)
           };

           Ok(GuardVerdict::Deny { reason })
       }
       ```

    3. **Create `read_structured_deny_reason` function:**
       ```rust
       fn read_structured_deny_reason(
           reason_fn: &wasmtime::TypedFunc<(i32, i32), i32>,
           memory: &Memory,
           store: &mut Store<WasmHostState>,
       ) -> Option<String> {
           const DENY_BUF_OFFSET: i32 = 65536;
           const DENY_BUF_LEN: i32 = 4096;

           // Call the guest's arc_deny_reason function
           let bytes_written = match reason_fn.call(store, (DENY_BUF_OFFSET, DENY_BUF_LEN)) {
               Ok(n) if n > 0 && n <= DENY_BUF_LEN => n,
               Ok(_) => return None,  // 0 or negative or too large
               Err(_) => return None, // call failed -- no reason
           };

           // Read the response from guest memory
           let mut buf = vec![0u8; bytes_written as usize];
           if memory.read(store, DENY_BUF_OFFSET as usize, &mut buf).is_err() {
               return None;
           }

           // Try to parse as JSON GuestDenyResponse
           match serde_json::from_slice::<crate::abi::GuestDenyResponse>(&buf) {
               Ok(resp) => Some(resp.reason),
               Err(_) => {
                   // Not valid JSON -- try as plain UTF-8 string
                   std::str::from_utf8(&buf).ok().map(|s| s.trim_end_matches('\0').to_string())
               }
           }
       }
       ```
