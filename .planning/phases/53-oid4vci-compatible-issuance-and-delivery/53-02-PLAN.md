# Plan 53-02

## Objective

Implement local and remote issuance-offer, token, and credential delivery
surfaces so operators can issue ARC passports through one concrete
interoperable flow.

## Work

- add CLI issuance commands that create offers and redeem delivered
  credentials
- add trust-control issuer metadata, token, and credential endpoints plus the
  authenticated operator path that creates offers
- persist offer redemption state durably enough to make pre-authorized
  issuance replay-safe and inspectable

## Verification

- local CLI roundtrip proving offer creation and credential redemption work
- remote trust-control regression coverage for metadata, token, and credential
  delivery
