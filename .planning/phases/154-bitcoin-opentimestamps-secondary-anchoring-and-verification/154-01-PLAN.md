# Plan 154-01

Implement OpenTimestamps publication preparation for checkpoint roots.

## Scope

- aggregate contiguous checkpoint roots into one super-root
- derive the OTS document digest for submission
- emit one explicit submission artifact with checkpoint range and calendars

## Success Criteria

- ARC can prepare deterministic OTS submission state for one checkpoint range
- the submission artifact names the exact super-root and digest being stamped
- empty or non-contiguous ranges fail closed
