# Phase 96: Portable Status, Revocation, Metadata, and Live Discovery Alignment - Context

## Goal

Converge portable credential lifecycle truth and live hosted metadata so ARC's
broader standards fabric has one fail-closed status and discovery story.

## Why This Phase Exists

ARC currently exposes truthful lifecycle and metadata surfaces, but the full
endgame needs standards-legible status, revocation, and live discovery
alignment before wallet, sender-constrained, and open-network work can widen
further.

## Scope

- portable status and revocation semantics
- issuer metadata and hosted authorization metadata alignment
- discovery freshness and status-drift behavior
- qualification fixtures for stale or contradictory lifecycle state

## Out of Scope

- public issuer/verifier discovery network
- open registry economics or governance
- external verifier-result federation
