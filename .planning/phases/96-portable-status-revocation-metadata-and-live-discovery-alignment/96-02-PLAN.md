# Plan 96-02

Align issuer metadata, credential metadata, and hosted authorization metadata.

## Scope

- reconcile portable issuer metadata with hosted resource and authorization metadata
- specify freshness windows, cache invalidation, and signing-key rotation effects
- deny stale or contradictory metadata combinations fail closed

## Success Criteria

- metadata surfaces have explicit consistency rules
- rotation and freshness behavior are testable
- consumers cannot rely on stale metadata silently
