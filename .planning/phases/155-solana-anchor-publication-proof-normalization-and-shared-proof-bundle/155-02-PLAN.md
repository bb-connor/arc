# Plan 155-02

Normalize EVM, Bitcoin, and Solana proofs into a shared bundle.

## Scope

- add one bundle wrapper over the primary proof and optional secondary lanes
- keep Solana evidence as a separate imported record within that bundle
- preserve the canonical anchor proof as the source of receipt truth

## Success Criteria

- one deterministic bundle format can represent the supported lane mix
- the primary proof remains canonical
- bundle verification stays reviewable lane by lane
